### Install
Use the [git](https://git-scm.com/downloads) to clone the project to your local machine.
```sh
$ git clone https://github.com/Testing-Ptoject.git
```

Navigate to the extracted folder
```sh 
cd Testing-Ptoject
```

Install the dependencies and modules
```sh
$ npm install
```

### Usage
```sh
$ npm start
```
